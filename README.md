### 📊 Bank Loan Dataset Analysis Dashboard
**Lending Club Dataset** | [GitHub](https://github.com/nikhil09012004) | *Aug '25 – Dec '25*

- Designed an interactive Power BI dashboard analyzing **38K+ loan records**, tracking KPIs such as **$435M funded**, **$473M received**, Avg IR ~12%, Avg DTI ~13%
- Evaluated portfolio risk through good vs bad loan analysis, identifying **86% low-risk** and **14% high-risk** loans
- Examined regional and temporal trends, highlighting **California & Texas** as top regions and **December** as peak application month
- Extracted borrower and loan-purpose insights, revealing **Debt Consolidation ~18K loans** as dominant and lower risk for long-tenure employees

**Tech Stack:** Power BI, DAX, Data Modeling, Data Visualization, KPI Metrics, Business Analytics

### 🤖 Loan Prediction System
**Loan Approval Dataset** | [GitHub](https://github.com/nikhil09012004) | *Jan '25 – Apr '25*

- Developed an end-to-end loan approval prediction system using Python analyzing **4,000+ loan records** to automate approval decisions
